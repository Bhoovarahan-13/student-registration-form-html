# 🎓 Student Registration Form

A simple and clean **Student Registration Form** built using HTML. This project demonstrates basic form handling and table structure in web development.

---

## 🚀 Features

- 📋 User-friendly registration form
- 📧 Email and password input fields
- 👤 Gender selection using radio buttons
- 🛠️ Skills selection using checkboxes
- 📅 Date of Birth picker
- 📊 Display submitted data in table format

---

## 🛠️ Technologies Used

- HTML5
